Courses

I participated & finished a fullstack web developer course from ZeroToMastery Academy.

In the course I was taught how to use scripting languages such as HTML5 & CSS3; I also learned how to use SASS & CSS preprocessors. I practiced a lot of JavaScript & was introduced to all the ECMAScript new features, like arrow functions, Async await, promises & callbacks to name a few.

Portfolio

Online Bookstore My first ever website was created as part of my graduation project. It was a functional online bookstore with a complete frontend/backend/database. I built the frontend with no dependencies, I built the backend with PHP & built the database using PHPMyAdmin.
Clinic website My second website was created to be an online portal for a clinic. It was purely frontend focused as I wanted to create a catchy design with a fluid & responsive UI. It was for a challenge I did on frontendMentor.com but I modified it to be a clinic instead of a bank website.
Budget app Created a compact CRUD budget tracking app using React. I used react-bootstrap for the styling. Implemented the use of localstorage to not lose state data if user leaves the site. This app is fully capable of adding, viewing & deleting budgets and/or expenses.
Face detection app Created a face detection app with its respective backend & database. I used react to handle states & react-bootstrap to handle styling. I built the backend with NodeJS & Express & implemented one-way password hashing through bcrypt. I used postgreSQL to handle the database & KnexJS to interact with it on the server. The app is powered with the Clarifai API.
Task Handler I created this CRUD project where a user can organize their tasks. This project was built with React, Typescript & Mongodb. I enforced user authentication with bcrypt password hashing & authorization with JSON web tokens. On the backend I integrated Helmet for securing headers & CORS for improved security.

This website was built with a lot of passion & is fully responsive.
